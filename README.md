
# Velvet Bite - Premium Restaurant Menu Website

## Features
- Responsive premium UI
- Dark & light mode
- Shopping cart with LocalStorage
- Favorites system
- Category filters
- Search functionality
- Toast notifications
- Parallax hero section
- Glassmorphism effects
- Hamburger mobile menu
- Animated loading screen
- Floating WhatsApp button

## Structure
- index.html
- style.css
- script.js
- assets/

## Usage
1. Extract the ZIP archive.
2. Open `index.html` in any browser.
3. Enjoy the fully functional restaurant menu website.

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
